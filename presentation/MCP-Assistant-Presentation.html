What is MCP Assistant?

A personal productivity platform that exposes 19+ AI-callable tools through the Model Context Protocol (MCP).

Users interact via a React chat interface backed by OpenRouter for multi-model LLM support. The assistant can check weather, manage notes and tasks, track time, set reminders, read news, and manage calendar events.

Two access paths: a local MCP server for IDE clients, and a multi-user Chat API for the web frontend.

MCP Server (port 8000)

Single-user, local IDE clients like Claude Desktop or Cursor

Chat API (port 8001)

Multi-user, per-user data isolation, React frontend

React Frontend (port 3000)

Modern chat UI with model switching and tools panel

Security & OWASP

Implemented Protections

Authentication & Isolation

✓ Per-user SQLite databases (data/{user_id}/)
✓ API keys hashed at rest (SHA-256)
✓ Re-login invalidates old key
✓ Auth required on all endpoints

Input & Query Safety

✓ Parameterized SQL everywhere
✓ Pydantic validation + max_length
✓ react-markdown sanitizes HTML (XSS)
✓ Localhost-only, CORS restricted

AI-Specific Safety

LLM Tool Guardrails

✓ READ/WRITE tool separation in system prompt
✓ Side-effect dedup (prevents duplicate writes)
✓ Tool loop capped at 5 rounds
✓ ToolAnnotations (readOnly/destructive hints)
